## Installation and Usage

Install the necessary packages to run the app

```bash
  npm install
```

## Running the App

Run the app with the following command and then scan de QR code.

```bash
$ expo start
```

Running the app in iOS

```bash
$ expo ios
```

Running the app in Android

```bash
$ expo android
```
